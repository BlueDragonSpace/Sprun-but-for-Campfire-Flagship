# Sprun but for Campfire Flagship

A turn-based strategy game with dimension-switching mechanics!

## Features

- Turn-based combat with multiple characters to control
- Action queue system and scaling mechanics
- Completely resizable to any resolution (built with Godot's Control Nodes!)
- 2D and 3D dimension switching for tactical advantage
- Player intents system
- 5 different enemy wave types
- Debuff system (Freeze, Poison, and more!)
- Cool animated background!

## Getting Started

This game is built with **Godot 4.6**. To run the project:

1. Install [Godot 4.6](https://godotengine.org/download) or later
2. Clone this repository
3. Open the project in Godot
4. Press F5 to run
