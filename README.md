# Sprun but Campfire Flagship

A dimension-switching turn-based tactical game built with Godot 4.6, where players switch between 2D and 3D combat to gain tactical advantages.

## About

**Sprun but Campfire Flagship** is the third iteration of the Sprun game project, currently being developed as part of Hack Club's Campfire Flagship program. This innovative game combines traditional turn-based strategy with a unique dimension-switching mechanic, allowing players to alternate between 2D and 3D perspectives to outmaneuver enemies.

## Key Features

- **Turn-based combat** with multiple characters to control, an action queue system, and dynamic scaling
- **Dimension-switching gameplay** - Switch between 2D and 3D turn systems to gain tactical advantages
- **Fully resizable** - Supports any resolution size using Godot's Control Nodes
- **Character intents system** - See what enemies are planning before they act
- **Multiple enemy wave types** - 5 different wave types to keep battles fresh
- **Debuff system** - Strategic status effects to weaken enemies
- **Cool dynamic background** and polished UI
